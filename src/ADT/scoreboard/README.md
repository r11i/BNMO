Cara Kompilasi Driver arraydin_sb_driver.c
1. Buka terminal
2. Ganti directory ke folder terletaknya ADT scoreboard
3. Jalani command ini: gcc arraydin_sb.c arraydin_sb_driver.c -o arraydin_sb_driver
